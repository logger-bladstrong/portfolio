如何不让熊大熊二捣乱

// Author: Zirui Zhang
// Category: Strategy
// Status: Solved

这是一个非常严肃的战术问题。

首先，我们需要分析“熊大”和“熊二”的行为模式。根据长期观察（参考《熊出没》相关文献），它们的主要动机通常是寻找食物（蜂蜜/水果）或保护森林。

策略一：蜂蜜诱导法（The Honey Pot Strategy）
在远离核心基地的区域设置高纯度蜂蜜诱饵。注意：必须定期补货，否则它们会因饥饿而回流。

策略二：光头强的反面教材
不要试图使用电锯或其他产生高分贝噪音的设备。这会直接触发仇恨值。

结论：
最好的防御是共存。只要不破坏森林，这两位通常是友好的NPC。
